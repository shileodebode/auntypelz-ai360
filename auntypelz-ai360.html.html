What is AuntyPelz AI 360?
AuntyPelz is Curacel's internally built people intelligence platform. It automates the 360-degree feedback cycle — reading raw peer feedback from Google Sheets, running GPT-4 analysis, and generating branded PDF reports complete with radar charts and competency scores. It has since expanded into a full web dashboard covering OKR management, role competency tracking, and Slack-triggered reminders. It is live, in active use, and the most evolved AI system in the People Ops function.

What Problem It Solves
⏱
Before AuntyPelz
  • Feedback synthesis was done manually by HR
  • Reports were inconsistent across cycles and reviewers
  • No structured competency scoring or visual output
  • OKR tracking was disconnected from feedback data
  • No audit trail for LLM performance or cost
✓
After AuntyPelz
  • Fully automated pipeline from raw data → branded PDF
  • Consistent AI synthesis across every employee and cycle
  • Radar charts, Start/Stop/Continue themes, competency scores
  • Web dashboard for HR, team leads, and admins
  • LLM observability via Langfuse (cost, quality, performance)
Core Capabilities
📄 360 Feedback Reports
Reads Google Sheets → GPT-4 synthesis → PDF with radar chart, competency scores, strengths, development areas, and Start/Stop/Continue themes. Sent via Gmail from peopleops@curacel.ai.
📊 360 Dashboard
Web UI for managing feedback cycles, viewing and deleting reports, and controlling team lead access. Role-based: Super Admin and Team Lead tiers.
🎯 OKR & Competency Tracking
Personal and team OKR management with appraisal workflows, finalization gates, and rating systems. Role competency management linked to Google Drive folder storage.

User Flows
HR: Generate 360 Reports
  1. Collect feedback via Google Sheets
  2. Admin triggers report generation for a period
  3. System filters for 4–5 minimum responses
  4. GPT-4 synthesises each person's feedback
  5. PDF generated with radar chart + branded format
  6. Report emailed — CC: peopleops@curacel.ai
Team Lead: Review OKRs
  1. Sign in to the dashboard
  2. Navigate to team OKRs drilldown
  3. View objective status, key result progress
  4. Manage priority and finalization gates
  5. Run appraisal workflows for rating periods
